# Glossary

Add a term once it appears in two or more places. Keep definitions to one or two sentences. Mark deprecated terms rather than removing them so older references still resolve.

Each entry uses the shape:

```markdown
**TERM**: one or two sentence definition. Cross-references in parentheses where useful.
```
